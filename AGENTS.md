# AGENTS.md

## Role

This repository is a public-facing portfolio implementation surface. Do not copy private wiki content, raw AI session logs, credentials, customer data, production endpoints, or internal hostnames into this repo.

## Work Rules

- Keep portfolio content screen-first and evidence-oriented.
- Label screen maturity honestly: implemented, prototype, planned, sanitized, archived.
- Prefer sanitized wireframes over raw admin/customer screenshots.
- Release notes may be drafted by LLM/Hermes but must be reviewed before public deployment.

## Validation

Run:

```bash
npm run typecheck
npm run validate:content
npm run build
```
