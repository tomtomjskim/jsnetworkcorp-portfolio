## Headline

제조 MES와 셀러형 커머스 운영 시스템 경험을 가진 백엔드 중심 풀스택 개발자

## Summary

PHP 기반 업무시스템에서 백엔드 기능 개발, 데이터 처리, 운영 화면, 관리자/셀러 업무 흐름을 다뤄 왔습니다. 제조 생산관리와 셀러형 커머스/물류 운영 도메인을 기반으로, legacy PHP 환경의 기능 고도화와 화면-API-데이터 흐름 정리를 병행했습니다.

## Full-stack Claims

- 업무 도메인의 화면, API, 데이터 흐름을 함께 고려한 기능 개선 경험 보유
- 제조 MES 및 커머스 운영 시스템에서 사용자 업무 흐름에 맞춘 기능 개선에 참여
- 셀러 상품, 입고, 재고, 외부출고, 대시보드 등 운영 화면과 백엔드 데이터 흐름을 다룸
- 파일 업로드, preview, confirm 기반 batch 처리 UX와 서버 검증 흐름을 함께 검토
- live-commerce를 신규 판매 채널로 보고 기존 커머스/fulfillment 구조와 연결 가능한 아키텍처 방향성 검토
- AI-assisted workflow를 활용해 설계 문서화, 테스트 기준 정리, 변경 이력 관리를 병행
