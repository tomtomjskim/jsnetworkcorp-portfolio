# Information Architecture

```text
/
├─ Home
├─ Projects
│  ├─ Product Engineering
│  ├─ AI Workflow / Agent Tooling
│  ├─ Infra / MCP / Safety
│  └─ Legacy Modernization
├─ AI Agents / LLM Portfolio
├─ Screens & Wireframes
├─ Resume
└─ Release Notes
```

The site is screen-first. Each project page starts with summary and visual evidence, then explains problem, approach, verification, and release status.

## Public Boundary

Do not publish private wiki contents, production logs, credentials, internal hostnames, customer data, order data, or raw AI session transcripts.
